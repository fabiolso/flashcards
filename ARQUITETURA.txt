/* 
   O projeto é uma Single Page Application (SPA) com múltiplas páginas HTML
   que compartilham CSS e JavaScript.
   
   Camadas:
   
   CAMADA 1: HTML (index.html, decks.html, review.html, manual.html)
   └─ Define a estrutura e elementos da página
   └─ Contém comentários explicativos sobre links entre páginas
   
   CAMADA 2: CSS (css/styles.css)
   └─ Estilos para TODOS os elementos
   └─ Mobile-first responsive (começa em 320px, escala até 1200px+)
   └─ Sem dependências externas
   
   CAMADA 3: JavaScript (js/*.js)
   ├─ js/utils.js      → Funções utilitárias comuns (localStorage, navegação, etc)
   ├─ js/audio.js      → Funções de áudio (text-to-speech via Web Speech API)
   └─ js/script.js     → Lógica principal (CRUD de cards, revisão, UI)
   
   CAMADA 4: LocalStorage
   └─ Armazena dados dos cards (não requer servidor)
   └─ Chave: "flashcards_cards"
*/

// ====================================
// 2. FLUXO DE DADOS
// ====================================

/*
   CRIAR CARD:
   
   user clicks "Criação de novo card"
         ↓
   abrirFormularioCriacao() [script.js]
         ↓
   Modal formulário aparece
         ↓
   user preenche campos + clica "Adesionar novo card / salvar"
         ↓
   salvarNovoCard(evento) [script.js]
         ↓
   validações (utils.js: estaVazio, limparTexto)
         ↓
   cria objeto card + ID único (utils.js: gerarID)
         ↓
   calcula proximaRevisao (utils.js: calcularProximaRevisao)
         ↓
   obterTodosCards() → array existente [script.js]
         ↓
   salvarTodosCards(cards) [script.js]
         ↓
   salvarNoLocalStorage(CHAVE_CARDS, cards) [utils.js]
         ↓
   localStorage.setItem() - BROWSER API
         ↓
   atualizarResumoProgresso() [script.js]
         ↓
   ✅ Card criado com sucesso
*/

/*
   REVISAR CARDS:
   
   user clica "Começar a revisão"
         ↓
   navegarPara('review.html') [utils.js]
         ↓
   review.html carrega
         ↓
   exibirProximoCard() [script.js]
         ↓
   carregarCardsParaRevisao() [script.js]
         ↓
   obterTodosCards() [script.js]
         ↓
   obterDoLocalStorage(CHAVE_CARDS) [utils.js]
         ↓
   localStorage.getItem() - BROWSER API
         ↓
   ordena por proximaRevisao (ASC = próximas primeiro)
         ↓
   mostrarCard(0) [script.js]
         ↓
   preenche elementos DOM:
      - reviewWordEnglish
      - reviewPhraseEnglish
      - reviewWordPortuguese
      - reviewPhrasePortuguese
   └─ mostra lado inglês (esconde português)
         ↓
   user pode clicar botões de áudio:
      - ouvirEmIngles() → ouvirFraseEmIngles() [audio.js]
         ↓
      - falarTexto(texto, 'en-US') [audio.js]
         ↓
      - window.speechSynthesis.speak() - WEB SPEECH API
         ↓
   user clica "Ver o cartão em Português"
         ↓
   mostrarLadoPortugues() [script.js]
         ↓
   mostra lado português, esconde inglês
         ↓
   user clica "Próximo card"
         ↓
   proximoCard() [script.js]
         ↓
   indiceCardAtual++
         ↓
   mostrarCard(novoIndice) [script.js]
         ↓
   repete... até último card
         ↓
   finalizarRevisao() [script.js]
         ↓
   navegarPara('index.html')
         ↓
   ✅ Revisão concluída
*/

   PÁGINA: decks.html

/*
   O SITE NÃO USA BANCO DE DADOS - TUDO É LOCAL!
   
   Chave Principal: "flashcards_cards"
   
   Estrutura armazenada:
   [
     {
       id: "1718192384920_0.8392",
       wordEnglish: "width",
       phraseEnglish: "Element width property",
       wordPortuguese: "largura",
       phrasePortuguese: "propriedade largura do elemento",
       dataCriacao: "2026-06-11T10:30:00.000Z",
       proximaRevisao: "2026-07-01T10:30:00.000Z"
     },
     {
       id: "1718192450123_0.5678",
       wordEnglish: "function",
       phraseEnglish: "A reusable block of code",
       wordPortuguese: "função",
       phrasePortuguese: "um bloco de código reutilizável",
       dataCriacao: "2026-06-11T10:45:00.000Z",
       proximaRevisao: "2026-07-01T10:45:00.000Z"
     }
   ]
   
   FLUXO SALVAR:
   ├─ Array de objects JS
   ├─ JSON.stringify(array) → string JSON
   ├─ localStorage.setItem(chave, stringJSON)
   └─ Dados persistem mesmo após fechar browser
   
   FLUXO CARREGAR:
   ├─ localStorage.getItem(chave) → string JSON
   ├─ JSON.parse(stringJSON) → Array de objects JS
   └─ Código pode usar dados como array
   
   LIMPEZA/RESET:
   ├─ limparLocalStorage() [utils.js]
   ├─ localStorage.clear() - DELETA TUDO
   └─ ⚠️ Use com cuidado!
*/

// 5. FUNÇÕES PRINCIPAIS POR ARQUIVO
// ====================================

/*
   ╔═══════════════════════════════════════════════════════════════╗
   ║ ARQUIVO: js/utils.js                                          ║
   ╚═══════════════════════════════════════════════════════════════╝
   
   NAVEGAÇÃO:
   └─ navegarPara(pagina) → window.location.href = pagina
   
   LOCALSTORAGE:
   ├─ salvarNoLocalStorage(chave, valor) → localStorage.setItem
   ├─ obterDoLocalStorage(chave, padrao) → localStorage.getItem
   ├─ removerDoLocalStorage(chave) → localStorage.removeItem
   └─ limparLocalStorage() → localStorage.clear (com confirmação)
   
   GERAÇÃO:
   └─ gerarID() → timestamp + random = ID único
   
   DATA/HORA:
   ├─ formatarData(data) → "11 de junho de 2026"
   ├─ diasDesde(data) → número de dias
   └─ calcularProximaRevisao() → Date +20 dias
   
   VALIDAÇÃO:
   ├─ estaVazio(texto) → boolean
   └─ limparTexto(texto) → sem espaços extras
   
   DOM:
   ├─ criarElemento(tag, class, atributos) → HTMLElement
   └─ copiarParaAreaTransferencia(texto) → clipboard
   
   DEBUG:
   ├─ logDebug(mensagem, dados)
   └─ logErro(mensagem, erro)
   
   ╔═══════════════════════════════════════════════════════════════╗
   ║ ARQUIVO: js/audio.js                                          ║
   ╚═══════════════════════════════════════════════════════════════╝
   
   ÁUDIO PARA REVISÃO:
   ├─ ouvirEmIngles() → document.getElementById + falarTexto
   ├─ ouvirFraseEmIngles() → document.getElementById + falarTexto
   └─ ouvirEmPortugues() → document.getElementById + falarTexto
   
   SÍNTESE DE FALA (Base):
   └─ falarTexto(texto, idioma, velocidade, volume, pitch)
      └─ window.speechSynthesis.speak() - WEB SPEECH API
   
   CONTROLE GLOBAL:
   ├─ pararAudio() → speechSynthesis.cancel()
   ├─ pausarAudio() → speechSynthesis.pause()
   └─ resumirAudio() → speechSynthesis.resume()
   
   DEBUG:
   └─ obterVozesDisponiveis() → array de vozes
   └─ listarVozesDebug() → console.log vozes
   
   ╔═══════════════════════════════════════════════════════════════╗
   ║ ARQUIVO: js/script.js                                         ║
   ╚═══════════════════════════════════════════════════════════════╝
   
   PÁGINA INDEX.HTML:
   ├─ abrirFormularioCriacao() → mostra modal
   ├─ fecharFormulario() → esconde modal
   ├─ salvarNovoCard(evento) → validar + salvar no localStorage
   ├─ abrirFormularioEdicao(id) → preenche formulário com card
   └─ atualizarResumoProgresso() → atualiza contadores
   
   GERENCIAMENTO DE CARDS:
   ├─ obterTodosCards() → obterDoLocalStorage [utils.js]
   ├─ salvarTodosCards(cards) → salvarNoLocalStorage [utils.js]
   └─ removerCard(id) → remove do array + salva
   
   PÁGINA DECKS.HTML:
   ├─ carregarListaCards() → renderiza todos os cards
   ├─ criarElementoCard(card) → cria HTML para um card
   └─ editarCard(id) → abrirFormularioEdicao
   
   PÁGINA REVIEW.HTML:
   ├─ exibirProximoCard() → carrega e mostra primeiro
   ├─ carregarCardsParaRevisao() → ordena por proximaRevisao
   ├─ mostrarCard(indice) → preenche DOM com dados
   ├─ mostrarLadoPortugues() → toggle visibility
   ├─ proximoCard() → ++indice + mostrarCard
   ├─ finalizarRevisao() → volta para index.html
   └─ atualizarContagemRevisao() → atualiza "X de Y"
*/

/*
   Este projeto usa APENAS tecnologias nativas do navegador (sem bibliotecas externas)
   
   APIs UTILIZADAS:
   
   1. localStorage API (armazenamento)
      ├─ window.localStorage.setItem(chave, valor)
      ├─ window.localStorage.getItem(chave)
      ├─ window.localStorage.removeItem(chave)
      └─ window.localStorage.clear()
      
      Documentação: https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage
   
   2. Web Speech API (áudio / text-to-speech)
      ├─ window.speechSynthesis (síntese de fala)
      ├─ window.SpeechSynthesisUtterance (objeto de fala)
      ├─ speechSynthesis.speak(utterance)
      ├─ speechSynthesis.cancel()
      ├─ speechSynthesis.pause()
      └─ speechSynthesis.resume()
      
      Documentação: https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API
   
   3. Date API (datas)
      ├─ new Date()
      ├─ Date.now()
      ├─ date.toISOString()
      ├─ date.toLocaleDateString()
      └─ date.getDate(), setDate(), etc
      
      Documentação: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date
   
   4. JSON API (serialização)
      ├─ JSON.stringify(objeto) → string
      └─ JSON.parse(string) → objeto
      
      Documentação: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/JSON
   
   5. DOM API (manipulação de página)
      ├─ document.getElementById(id)
      ├─ document.createElement(tag)
      ├─ element.textContent = valor
      ├─ element.classList.add/remove/toggle()
      └─ element.addEventListener(evento, funcao)
      
      Documentação: https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model
   
   6. History API (navegação)
      ├─ window.location.href = url (navegar)
      └─ window.location.reload() (recarregar)
      
      Documentação: https://developer.mozilla.org/en-US/docs/Web/API/History_API
*/
